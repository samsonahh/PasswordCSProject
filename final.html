  • Files

    • final.html: This html file
    • main.html: The main landing page for the project, containing the search form.
    • search.py: Python prgram that is the action for the form in main.html. Reposinsible for parsing the quesry string, loading the cards into a dictionary and responding to the query. Calls functions in page_maker.py
    • page_maker.py: Python program that generates the html pages for single or multiple card search results
    • swlcg_cards_full.csv: csv file of all cards
    • card_images/*: Directory containing images for each card, organized by a unique id number, These images were not uploaded to the homework server.

  • Bugs, Errors and Things to be Aware of

    • Searching by name must be an exact name match (aside from capitalization).
    • If you try to seach by multiple categories, only the first one with data will be used.
    • A maximum of 100 cards will be displayed, even if there are more matches.
