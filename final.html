How it works

Files

  • final.html: This is the html file that contains the documentation of our project
  • 10k-most-common.txt: This file contains a list of the most common words and number combinations.
  • words_alpha.txt: This file contains a list of English dictionaries words.
  • passchecker.py: This is the python program that checks whether your inputted password is strong and generates a random password that is strong.

Resources

  • Card information: http://www.cardgamedb.com/index.php/starwars/star-wars-card-spoilers
  • Card images: http://picsofswlcgcards.com

Bugs, Errors and Things to be Aware of

  • Searching by name must be an exact name match (aside from capitalization).
  • If you try to seach by multiple categories, only the first one with data will be used.
  • A maximum of 100 cards will be displayed, even if there are more matches.
